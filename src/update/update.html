Whats new?

  • version 2.0.2 (11th November 2013)
    • Fixed issue where margin against the document.body would cause the ruler position to be offset by that amount
    • Element positions and sizes are now more accurately calcuated using getBoundingClientRect()
    • Re-styled the number input spinner buttons so they work on sites that hide them
    • Various other minor CSS updates to prevent clashing with page styles
    • [translation] Added Vietnamese translation thanks to Khoi Nguyen (soundcloud.com/potasmic)
  • version 2.0.1 (3rd November 2013)
    • [issue #1] Fixed bug that caused the close () button on the toolbar to not work
    • Fixed display issue if the website used -webkit-appearance:none
    • Fixed occasional display issue with the tracking mode toggle button
    • Added popup error messages when trying to use the extension on the chrome web store or local chrome pages
    • Added new update page containing changelog and contributors
  • version 2.0.0 (18th October 2013)
    • Brand new version rewritten from the ground up!
